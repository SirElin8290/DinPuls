# DinPuls.se

Ren, lättredigerad prototyp för Åmåls kommun.

## Starta
1. Öppna mappen i Visual Studio Code.
2. Högerklicka på `index.html`.
3. Välj **Open with Live Server**.

## Struktur
- `index.html` – sidans innehåll
- `styles.css` – designen, uppdelad i tydliga avsnitt
- `script.js` – enkel interaktivitet
- `assets/amal.jpg` – Åmålsbild
- `assets/logo.png` – preliminär logga
